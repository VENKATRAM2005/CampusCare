# 🎓 CampusCare

> Intelligent campus grievance management platform built using **Spring Boot**, **React**, and **PostgreSQL**, featuring AI-assisted complaint analysis and role-based workflows.

> 🚧 **Project Status:** Actively under development

---

## 🌟 Overview

CampusCare is a full-stack web application designed to modernize grievance handling in educational institutions. It enables students to submit complaints, mentors and HODs to manage departmental issues, and administrators to monitor institutional activity through centralized dashboards.

The project focuses on clean architecture, secure authentication, and AI-assisted complaint analysis while following modern full-stack development practices.

---

## ✨ Current Features

- ✅ JWT Authentication
- ✅ Role-Based Access Control
- ✅ Student Complaint Portal
- ✅ Complaint Tracking
- ✅ AI-assisted Complaint Analysis (Gemini)
- ✅ Spring Boot REST APIs
- ✅ PostgreSQL Integration
- ✅ React + TypeScript Frontend
